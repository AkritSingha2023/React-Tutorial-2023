Now the question is why react?

The answer is very simple. For a browser the costly and critical task is to render html on runtime. Runtime means when you have clicked any button, certian thing changes in html. At that time, the browser will perform the most costly processing - DOM manipulation. It manupulates the DOM tree and updates the structure.
So, meny tech gaints invents their algorithms to optimize the DOM manipulation process. And as we know React comes with the optamized algorithms. That's why we are here to learn React, React powers.

Intersting right!
React Developers submit two files. react.development.js - It comes with core react algorithms. You may heard about react native, used in App. So this file is the common file for both platforms. react-dom.development.js - This file is responsible for DOM manipulation. It's here for interact with browser, not used in App.

So, here is the example - how we can create element and insert element in React. This looks similar to js.

  • Create Element -

    In React, the core file named react.development.js it gives an object React. It has a function named createElement which do the similar thing like document.createElement. It takes three parameters: 1. the html tag, 2. an object, 3. inner text


  • Define Root Element -

    react-dom.development.js gives an object ReactDOM. It has a function named createRoot which helps to select an element, similar like document.getElementById.


  • Insert header to the root element -

    ReactDOM has another function named render which helps insert one element to another element, smae like appendChild.
